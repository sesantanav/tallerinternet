¿Qué es bootstrap?

Descripción de bootstrap, may 7, 2021

Es un conjunto de herramientas de código abierto para el desarrollo de sitios y páginas web responsive con HTML, CSS y JavaScript. Brinda una serie de elementos básados en plantillas de CSS y JavaScript que podemos utilizar de manera sencilla. Consta de diferentes componentes, tales como, ventanas modales, menús, botones, formularios, entre otros. Todo lo necesario para maquetar un sitio web.

Su gran ventaja es que las interfaces que es posible crear con bootstrap son adaptables a todo tipo de dispositivos y pantallas.
